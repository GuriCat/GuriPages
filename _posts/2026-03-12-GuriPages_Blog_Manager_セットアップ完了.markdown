GuriPages の blog を Genspark AI から自動管理できるようになりました。

## できること

- ➕ 新規エントリーの追加
- ✏️ 本文の編集
- 🔧 タイトル・タグ・日付の修正
- 🗑️ エントリーの削除

## 使い方

パスフレーズを伝えるだけで、AI が自動的に clone → 編集 → push まで行います。

```
「〇〇というテーマで新記事を書いて。パスフレーズは koji98lp@」
```
